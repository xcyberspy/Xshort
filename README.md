<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Shorten long URLs with custom aliases
*   Check alias availability
*   URL history tracking
*   Clean modern GUI with customtkinter

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository:</p>

```
git clone https://github.com/xcyberspy/Xshort.git
```

<p>2. Navigate to the Directory</p>

```
cd Xshort
```

<p>3. Install Required Packages</p>

```
pip install customtkinter requests pillow
```

<p>4. Set up your Bitly API token [Add: BITLY_TOKEN="YOUR API TOKEN HERE"</p>

```
https://app.bitly.com/settings/api
```

<p>5. Run the application</p>

```
python app.py
```
